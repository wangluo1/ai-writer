# AI Writer

Select any text on any webpage → polish or translate instantly with AI.

## Installation

### Chrome / Edge / Brave
1. Download `ai-writer-extension.zip` from [Releases](https://github.com/yourusername/ai-writer/releases)
2. Open `chrome://extensions` → enable Developer mode
3. Drag the zip file onto the page

Or install from [Chrome Web Store](#) (coming soon).

### PopClip (macOS)
```bash
curl -fsSL https://raw.githubusercontent.com/yourusername/ai-writer/main/popclip/install.sh | bash
```
Then select any text anywhere on macOS → PopClip menu → 润色 or 翻译.

## Features

- **Select any text** on any webpage → floating menu with Polish & Translate
- **3-Agent pipeline**: Writer drafts → Checker reviews → Editor finalizes (Chrome extension)
- **Chinese ↔ English translation** with natural, native-like output
- **Supports DeepSeek, OpenAI, and custom API** endpoints
- **Keyboard shortcuts**: ⌘E to polish, ⌘T to translate in any text field
- **Bring your own API key** — no data collection, no monthly fees

## How it works

1. Configure your API key (DeepSeek recommended, ~¥0.002/request)
2. Select text anywhere
3. Click Polish or Translate

All API requests go directly from your browser to your provider. No middle servers.
